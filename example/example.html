介绍

Wood UI是一款Javascript打造的UI插件库，她小巧、简洁，体积不到 50KB

她可以被用来构建你的静态页面，目前为止共包含组件16个，可以实现快速的页面搭建。现在，仍在持续开发中

本页面是Wood UI的文档和使用说明，这里是源代码地址

无论你是拿来使用还是拿来二次开发，都非常欢迎！

安装

目前， Wood UI 只提供了外部文件引入的方式，后期，等稳定版本开发完成就会推出模块化的引入方式

首先需要下载两个文件，点击这里

使用

Wood UI会挂载一个名为 wdu 的变量到window上，然后就可以按需调用组件了

首先需要在html中定义好元素，然后加上相应的组件的类名，接着通过 data-* 传入配置项

最后，保证 woodui.js 加载以后，在一个script标签中，通过　wdu.组件类名 来实例化相应的组件，例如：const btn = new wdu.Button()

示例

下面给出一个完整的示例，共分为 4 个步骤

          <!DOCTYPE html>
          <html lang="en">
          <head>
          <meta charset="UTF-8">
          <title>Document</title>

          <!-- 1. 引入 CSS 样式文件 -->
          <link rel="stylesheet" href="./dist/woodui.css">
          </head>
          <body>
          <div>
          <div class="wdu-box">

          <!-- 4. 在 html 中定义 wdu 元素 -->
          <button class="wdu-button"
          data-option='{"type":"normal"}'>确认</button>
          </div>
          </div>

          <!-- 2. 引入 js 脚本 -->
          <script src="./dist/woodui.js"></script>
          <script>

              // 3. 按需调用，new 相应的组件对象
              const btn = new wdu.Button()
          </script>
          </body>
          </html>

用法： Message用于在页面上弹出一条消息，提示用户，调用方式为 wdu.Message()，接受一个对象 config 作为参数，其中有四个属性

            title: '重要提示'
            content: '一些文本内容或其他内容'
            event: ()=>{alert('执行回调函数')})
          })
